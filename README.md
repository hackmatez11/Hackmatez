# MedCare

##  Project Overview

MedCare is a **full-stack healthcare management system** designed to digitize and simplify healthcare services. The project focuses on improving patient–doctor interactions, managing healthcare records, and providing modern features such as QR-based access and intelligent assistance.

This project is suitable for **academic final-year projects, capstone projects, and real-world prototypes** in the healthcare domain.

---

##  Key Features

*  Patient and user management
*  Healthcare service management
*  Digital medical records
*  Secure authentication and authorization
*  QR code–based functionality
*  Intelligent chatbot / RAG bot support
*  Web-based user interface

---

##  Technology Stack

### Frontend

* React.js

### Backend

* Node.js

* QR code generation & scanning

##  Installation & Setup

###  Clone the Repository

```bash
git clone https://github.com/hackmatez11/Hackmatez.git
cd HealthcareFinal
```

###  Backend Setup

```bash
cd backend
npm install
node server.js
```

The backend server will start on:

```
http://localhost:5000
```

###  Frontend Setup

```bash
cd Healthcare
npm install
npm run dev
```

The frontend will run on:

```
http://localhost:3000
```

---

##  Environment Variables

Create a `.env` file in the backend folder and add:

```env
PORT=5000
Supabase_secret_key =your_database_access_string
Supabase_access_key =your_database_access_string

```
